# Primary Key

## Definition

A Primary Key is a field that uniquely identifies every record in a table.

## Roman Urdu

Primary Key har record ki apni unique identity hoti hai.

Bilkul CNIC ki tarah.

Do logon ka CNIC kabhi same nahi hota.

Isi tarah database mein Primary Key kabhi duplicate nahi hoti.

## Example

| StudentID | Name |
|-----------|------|
|101|Ali|
|102|Sara|
|103|Ahmed|

Primary Key = StudentID

## Rules

- Must be Unique
- Cannot be NULL
- One Primary Key per table

## Cambridge Exam Tip

Exam mein aksar poocha jata hai

Why is StudentID a better Primary Key than Name?

Answer

Because names can repeat but StudentID cannot.
